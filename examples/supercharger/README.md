# Supercharger Fixtures

Compact JSON fixtures for advanced Structure Factory orchestration patterns: active learning, screening shards, scientific memory, evidence graphs, receptor-state registries, and design tranche planning.

## Use It For

- schema and adapter examples without real provider runs
- agent planning around screening and design fanout
- public-safe evidence-graph and active-learning demonstrations
- fixture inputs for `make screening-schema-check`

## Hand A Mission To An Agent

```text
Use the BioSymphony Structure Factory skill. Review examples/supercharger as public-safe orchestration fixtures. Explain what each fixture demonstrates, keep every claim at public_synthetic_demo or planning level, and treat the JSON files as contract shape rather than real biological results.
```

## Run It Yourself

```bash
make screening-schema-check
bsf audit .
```

The files in this folder are examples of contract shape. Real screening output, generated candidate sequences, private biological data, provider logs, and model weights live in operator-controlled infrastructure outside the repo.
