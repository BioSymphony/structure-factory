# Skills

This folder contains portable agent skills.

## BioSymphony Structure Factory

Path:

```text
skills/biosymphony-structure-factory/SKILL.md
```

Use this skill for public-safe structural biology campaign planning, binder-design triage, model-jury dossiers, RunPod/cloud stage contracts, and tracker-neutral issue packs.

Install into a local Codex skill directory:

```bash
scripts/install-codex-skill.sh
```

Validate after edits:

```bash
make harness-check
```

The portable skill and `.codex/skills/biosymphony-structure-factory/SKILL.md` must stay byte-identical.
