# Internal Notes

This directory defines the private-side note pattern for Structure Factory.

Tracked files here may contain templates and guidance only. Local operator notes, private image names, accepted-license notes, local install paths, provider account assumptions, and testing decisions belong under:

```text
internal/private/
```

`internal/private/` is gitignored. It is for local convenience only and must still not contain API keys, tokens, license files, license IDs, private installer URLs, private biological data, raw cryo-EM data, model weights, or unpublished sequences.

Use `internal/templates/structure-factory-testing-notes.template.md` when creating private testing notes.
