# md-docking Image Plan

## Purpose

Molecular dynamics, docking, and structure-analysis support for model-comparison and validation campaigns.

## Candidate Contents

- OpenMM
- GROMACS
- AutoDock Vina
- GNINA
- RDKit
- MDAnalysis
- MDTraj
- PLIP or interaction-analysis helpers where appropriate

## Smoke Command

Run toolcheck and GPU visibility checks only. No receptor/ligand datasets are downloaded during prep.
