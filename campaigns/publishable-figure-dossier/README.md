# Publishable Figure Dossier

## Objective

Create publication-grade structural figures with reproducible render scripts, sessions, captions, provenance, and validation notes.

## Required Dossier

```text
figure-dossier/
  figure_manifest.json
  inputs/
  panels/
  sessions/
  scripts/
  captions/
  provenance.md
  visual_qa.md
```
